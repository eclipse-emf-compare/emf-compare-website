Textual Comparison for multi-line attributes

Matching Strategies

Two main families of strategies are supported by EMF Compare : matching elements by ID or using the elements data. If ID's are provided, it will use them by default but you can change this behavior using the preferences page.

High Level Diff

Sometime it might be usefull to hide a set of model differences behind a unique differences which makes more sense for the end user. The components allows you to do so with the "Diff Extension" extension point.
